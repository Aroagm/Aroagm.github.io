Final project Master Data Science Kschool

Objective of the project

The purpose of this project is to apply all the techniques and skills acquired during the Data Science Master from Kschool.  
I chose a topic related to my study field, energy, in particular I decided to work with household consumption data. The objective is to develop a tool that helps the energy audit of a home. The output is a recommendation of slight changes in periods to use certain electrical household appliances.

Data sources

For the this project there are two datasets that have been used, to access the sources simply click on the image

  • Household data containing diferent load information (kWh) in hourly resolution
  • Hourly prices (€/MWh)
